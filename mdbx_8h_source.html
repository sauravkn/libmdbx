611  * - Modern Windows versions, including Windows Vista and later, provides
612  * support for "TLS Directory" (e.g .CRT$XL[A-Z] sections in executable

620  * so the MDBX_MANUAL_MODULE_HANDLER defined as 1.

624  * the MDBX_MANUAL_MODULE_HANDLER. */
